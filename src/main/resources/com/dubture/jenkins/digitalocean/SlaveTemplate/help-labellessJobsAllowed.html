If there is non-empty set of labels specified and this is checked, jobs that are not restricted to run on any specific node will trigger creation of a Droplet based on this Slave Template.

If there is non-empty set of labels specified and this is not checked, jobs that are not restricted to run on any specific node will not trigger creation of a Droplet based on this Slave Template.

If no labels for this Slave Template are specified, then regardless of whether this option is checked or not, jobs that are not restricted to run on any specific node will trigger creation of a Droplet based on this Slave Template.
